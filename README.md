# Exámenes de PHP

Este repositorio contiene una recopilación de mis exámenes y prácticas de **PHP**, realizados a lo largo del curso :)

## Contenido

Aquí podrás encontrar distintos ejercicios y exámenes que cubren temas como:

* Conexión a bases de datos con PDO
* CRUD (Crear, Leer, Actualizar, Borrar)
* Manejo de formularios
* Validación de datos
* Uso de sesiones
